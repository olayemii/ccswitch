---

## Requirements

- Node.js 18 or later
- macOS, Windows, or Linux

## Install from npm

```bash
npm install -g @olayemii/ccswitch
```

## Install from source

```bash
git clone https://github.com/olayemii/ccswitch.git
cd ccswitch
npm install
npm run build
npm link
```

This puts `ccswitch` on your `PATH`.

## Verify

```bash
ccswitch --version
```
